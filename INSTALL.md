# CleanWeb Extension - Installation Guide

## Quick Start

1. **Build the Extension**
   ```bash
   npm install
   npm run build
   ```

2. **Install in Chrome**
   - Open Chrome
   - Go to `chrome://extensions/`
   - Turn on "Developer mode" (top right toggle)
   - Click "Load unpacked"
   - Select the `dist` folder

3. **Test the Extension**
   - Navigate to any website
   - Click the CleanWeb icon in your toolbar
   - Click "Entrer en mode Ã©dition"
   - Hover over elements and click to hide them
   - Press Escape to exit selection mode
